Privacy Policy

Last Updated: 12/11/2025

This Privacy Policy explains how Lonar ("the Application", "we") collects, uses, and protects the personal data of users ("User") in compliance with Google API Services User Data Policy and applicable privacy laws.

The Application is a personal assistant designed to help Users manage and automate personal and professional tasks within Google services (such as Gmail, Google Drive, Google Calendar, and Google Tasks).

1. Data Collected and Purpose of Use

The Application requires access to the following sensitive data (defined as "Sensitive Scopes" by Google) solely to provide the requested functionality:

Google Service Data Collected Purpose of Use (CRITICAL)
Gmail Read, send, delete, modify emails Only to perform actions requested by the User, e.g., "Send an email to Mario", "Summarize unread emails", "Find flight receipt X".
Google Drive Access, modify, create, delete files Only to retrieve or create documents on User request, e.g., "Find March report" or "Create a new note in Drive".
Google Calendar Access, create, modify events Only to manage the calendar, e.g., "Schedule a meeting tomorrow at 10:00" or "Check my appointments".
Google Tasks & Contacts Manage tasks and search contacts Only to add reminders or find necessary information to perform actions, e.g., sending emails.

IMPORTANT NOTE: The Application only performs actions explicitly requested by the User and does not access data for any purposes other than those described.

2. Prohibition of Unauthorized Use

  • NO THIRD-PARTY SHARING: User sensitive data (Gmail, Drive, Calendar, Contacts) will never be shared, sold, or rented to third parties for marketing, advertising, or unrelated purposes.
  • NO ADVERTISING: The Application does not display ads and does not use personal data for advertising purposes.
  • NO USE BEYOND FUNCTIONALITY: Data is used only to process User commands and provide expected responses.

3. Data Security and Retention

All data exchanged between the Application and Google servers is handled via encrypted connections (HTTPS). The Application does not store full email contents or Drive files on its servers. Session data necessary for operation (such as access tokens) is protected and revocable at any time.

4. User Rights

Users have the right to revoke access to the Application at any time via their Google account security settings. Revocation will immediately terminate all services provided by the Application.
